# Usage

## Basic Import

Import components from the main package:

```jsx
import { DrawerLayout, Header, Label, ValueCard } from "@pautena/react-design-system";
```

## Styles

Import the shared stylesheet once in your app entrypoint:

```jsx
import "@pautena/react-design-system/global.css";
```

## Component Examples

### Data Display

```jsx
import { Label, ValueText, ValueCard } from "@pautena/react-design-system";

function UserProfile() {
  return (
    <>
      <Label text="Active" preset="success" />
      <ValueText label="Email" value="user@example.com" />
      <ValueCard title="Revenue" value="$1,234" />
    </>
  );
}
```

### Layouts

```jsx
import { DrawerLayout } from "@pautena/react-design-system";

function Dashboard() {
  return (
    <DrawerLayout
      header={{ title: "Dashboard" }}
      menu={[
        { label: "Home", onClick: () => {} },
        { label: "Settings", onClick: () => {} },
      ]}
    >
      {/* Your page content */}
    </DrawerLayout>
  );
}
```

## Testing Utilities

Import testing helpers for unit tests:

```jsx
import { render, screen } from "@pautena/react-design-system/testing";
import { Label } from "@pautena/react-design-system";

test("renders label", () => {
  render(<Label text="Active" />);
  expect(screen.getByText("Active")).toBeInTheDocument();
});
```

The testing export includes:
- Pre-configured render for design system components
- All exports from `@testing-library/react`
- Custom testing utilities
